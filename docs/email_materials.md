# Email Materials

## Short Project Description

Open-World Minecraft Agent Lab is an unofficial student research repository for studying Minecraft open-world agents through reading notes, repository audits, bounded feasibility checks, and lightweight toy demos.

## One-Sentence Research Interest

I am interested in how planning, memory, visual grounding, and action representation connect in embodied Minecraft agents.

## Main Story

```text
planning -> memory -> visual grounding -> action prediction
```
