# AutoDL Setup

## Repository Setup

```bash
git clone https://github.com/Seren666/openworld-minecraft-agent-lab.git
cd openworld-minecraft-agent-lab
bash scripts/setup_autodl_env.sh
```

## Storage Rules

- Put large datasets under `data/` or `datasets/`.
- Put checkpoints under `checkpoints/`, `ckpts/`, or `models/`.
- Put generated outputs under `outputs/`, `runs/`, or `autodl-tmp/`.
- Keep small Markdown summaries in `experiments/`.
