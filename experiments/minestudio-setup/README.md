# MineStudio Setup

This experiment records MineStudio installation, environment setup, and first-run notes.

## Goals

- Document local setup requirements.
- Document AutoDL setup requirements.
- Record errors, fixes, and reproducible commands.
- Keep small screenshots that clarify setup results.

## Files

- `install_notes.md`: installation commands and dependency notes
- `run_logs.md`: small Markdown summaries of runs
- `screenshots/`: small setup screenshots
