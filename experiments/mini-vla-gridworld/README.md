# Mini VLA Gridworld

This experiment is a lightweight sandbox for vision-language-action ideas before running heavier Minecraft experiments.

## Goal

Build a small gridworld where an agent receives a visual state, a language instruction, and must produce an action sequence.

## Planned Questions

- How should actions be represented?
- How much history is needed for reliable instruction following?
- Which failures resemble Minecraft planning or control failures?

## Directory Layout

- `src/`: source code for the sandbox
- `configs/`: small experiment configs
- `results/`: small summarized results
